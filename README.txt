-------------------------------------

To run SpikeX, first add this folder and subfolders to path by clicking the Matlab HOME tab, then setpath.

Run main.m (calls gui.m).    Set sampling rate first, then load data file, e.g. a file from the hc-1 dataset.

See Readme_screenshots in this folder.
